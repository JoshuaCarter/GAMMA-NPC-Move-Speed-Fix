# Dorn's NPC Move Speed Fix v0.0.0

In **S.T.A.L.K.E.R. G.A.M.M.A.**, stalker NPCs can feel like their player eurobeat with how fast they move and strafe in combat. This mod drops selected `[stalker_movement_speeds]` values so NPC movement feels more natural.

## What this mod does

- Lowers stalker NPC combat move speeds
- Leaves ISG (UNISG) on vanilla stalker move speeds, overriding G.A.M.M.A. NPCs Faster Reactions' boosted ISG speeds

## Installation

1. Download the latest release from https://github.com/JoshuaCarter/GAMMA-NPC-Move-Speed-Fix/releases
2. Install with Mod Organizer 2 like any other mod
3. Place this mod **below** mods that override `m_stalker.ltx` (e.g. G.A.M.M.A. NPCs Faster Reactions, Stealth Overhaul)
